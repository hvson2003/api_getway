## Các API Endpoint có sẵn

### **Product Service**
_(Thông qua API Gateway - port 8080)_

- **GET** `/api/products`  
  _Lấy tất cả sản phẩm_

- **GET** `/api/products/{id}`  
  _Lấy sản phẩm theo ID_

- **POST** `/api/products`  
  _Tạo sản phẩm mới_

---

### **Order Service**
_(Thông qua API Gateway - port 8080)_

- **GET** `/api/orders`  
  _Lấy tất cả đơn hàng_

- **GET** `/api/orders/{id}`  
  _Lấy đơn hàng theo ID_

- **POST** `/api/orders`  
  _Tạo đơn hàng mới_
